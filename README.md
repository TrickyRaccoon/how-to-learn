Научиться учиться 

"Научиться учиться" это сайт в котором собрана информация о том как эффективно обучаться. Сайт спроектирован так, чтобы пользователь смог понять и разобраться в проблемах современного обучения и выбрать удобные ему практики. Сайт был сверстан с помощью CSS3 и HTML5. Для файловой структуры и семантики была выбрана методология БЭМ.Для реализации идей проекта была выбрана блочная структура сайта и задействованы  такие технологии построения сетки как "flexbox" и "grid". Для импорта видео использовалась технология "iframe". Для анимации фоновых картинок использовалась технология "keyframes".

Список возможных доработок

    2) Добавить больше образовательных видео в блок video.

    4) Дописать вендорные префексы для кроссбраузерности. 
    5) Форма для комментариев от пользователей.
    6) Сделать анимацию в блоке feynman
